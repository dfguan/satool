# satool
satool is a tool for managing SAT format. 

## Dependencies
1. zlib

## Installation

Please use the following code to install satool:

```
git clone https://github.com/dfguan/satool.git
cd satool/src && make
```

## Usage
### Convert SAT to AGP
Given a **sat** file, convert it into AGP format
```
bin/satool agp $sat > t.agp
```
